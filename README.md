# Cadastro-de-produtos-com-mongoDB
Projeto desenvolvido para aplicação do conhecimento adquirido da integração do Node.js com MongoDB utilizando o ODM Mongoose.
### Tecnologias utilizadas
* Plataforma Node.js
* Express.js
* Handlebars
* MongoDB (ORM Mongoose)
